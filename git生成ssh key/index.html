# git生成部署

# mac 版

mac 下生成 git ssh key 由于本地 Git 仓库和 GitHub 仓库之间的传输是通过 SSH 加密的，所以必须要让 github 仓库认证你 SSH key，在此之前，必须要生成 SSH key。

# 第一步：

cd ~/.ssh 检查本机的 ssh 密钥

# 第二步：

ssh-keygen-t rsa -C "邮件地址@youremail.com" 例: ssh-keygen -t rsa -C'55343581@qq.com';

# 第三步：

.ls 显示目录名 .cat id_rsa.pub 显示 id_rsa.pub 文件内容复制在 github ssh 设置里面粘贴复制的内容 .测试：在终端输入 ssh -T git@github.com

# Window 版

windows 下生成 git ssh key 由于本地 Git 仓库和 GitHub 仓库之间的传输是通过 SSH 加密的，所以必须要让 github 仓库认证你 SSH key，在此之前，必须要生成 SSH key。

# 1、cd ~/.ssh/

【如果没有对应的文件夹，则执行 mkdir ./.ssh】

# 2、输入 git 用户名及邮件

git config --global user.name "你的用户名"

git config --global user.email "你的公司或个人邮箱"

# 3、打开 Git Bash

输入 ssh-keygen -t rsa -C "yourEmail@email.com"，”email@email.com”是你的 github 账号.

# 4、打开目录

此目录下会生成id_rsa和sid_rsa.pub文件

id_rsa 中的内容便是私钥，id_rsa.pub 中的内容便是公钥。

# 5、用记事本打开 id_rsa.pub 文件，复制内容，

在 github.com 的网站上到 ssh 密钥管理页面，添加新公钥，随便取个名字，内容粘贴刚才复制的内容。

# 6、完成设置。
